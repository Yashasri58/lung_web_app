# Lung Cancer Prediction Web App

This project is a Flask web application that trains multiple machine learning models to predict lung cancer risk from user inputs.

## Features
- Multiple model comparisons
- Probability-based prediction output
- Visualization charts for model performance and feature importance

## Run locally
1. Create a virtual environment
2. Install dependencies
3. Run `python app.py`

## Notes
The app uses the dataset in `cancer.csv` and serves charts from the `static/` directory.
